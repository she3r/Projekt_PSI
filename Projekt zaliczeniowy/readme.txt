## Projekt zaliczeniowy PSI

### Autor: Piotr Kubacki
### Zbior danych: https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset
(Predicting Stroke Dataset)
### Temat: Klasyfikacja: Przewidywanie zawału u pacjenta
### Jezyk: Python

## Po wstępnej kwalifikacji, wybrano modele do uczenia:
- MLPClassifier
- SVC linear
- SVC rbf
- Logistic Regression
- AdaBoost
Każdy z nich wygenerował podobny, bardzo dobry wynik. Nieco lepszy jednak uzyskał MLP Classifier: 0.95555 acc
